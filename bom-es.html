Componentes Requeridos

Interruptor Inteligente TP-Link Kasa
~$20

Cantidad: 1

Interruptor inteligente habilitado para WiFi con control físico. Reemplazo directo para el interruptor de palanca existente.

Modelo: HS200 (o KS225 si se necesita atenuación)

Características: WiFi, App, Voz, Programación, Compatible con Google Home y Alexa

Conectores de Cable
~$3

Cantidad: 1 paquete

Conectores de cable surtidos para conexiones seguras. Probablemente ya presentes, pero es bueno tener repuestos.

Herramientas y Equipo de Seguridad Requeridos

Probador de Voltaje Sin Contacto
~$15

Cantidad: 1 - CRÍTICO PARA SEGURIDAD

Verificar que la corriente está APAGADA antes de tocar cualquier cable. Herramienta de seguridad no negociable.

⚠ CRÍTICO: Nunca trabajes en electricidad sin esta herramienta. Prueba CADA cable antes de tocar, incluso después de apagar el interruptor.
Juego de Destornilladores
~$12

Cantidad: 1 juego

Phillips y plano para remover la placa del interruptor y terminales de cableado.

Linterna Frontal o Linterna
~$10

Cantidad: 1 - RECOMENDADO

El interruptor estará apagado, así que las luces de la habitación no funcionarán. La iluminación con manos libres es crucial.

Requisitos de Software y Configuración

Aplicación Kasa Smart (Gratis)

Requerida para configuración inicial de WiFi. Después de la configuración, se puede controlar vía Google Home exclusivamente.

Aplicación Google Home (Gratis)

Interfaz de control principal para automatización, programación y comandos de voz.

Red WiFi

WiFi de 2.4GHz requerido en la ubicación del interruptor. Los interruptores inteligentes no soportan 5GHz.

COSTO TOTAL DEL PROYECTO (Solo Artículos Requeridos)
~$50-60

Interruptor + Herramientas de Seguridad + Conectores de Cable
Inversión única • Herramientas reutilizables para proyectos futuros

⚠ Antes de Comprar

Verificar presencia de cable neutro: Remover la placa del interruptor (corriente APAGADA) y confirmar que hay un conjunto de cables blancos en la caja. El Kasa HS200 requiere neutro.

Si NO hay cable neutro presente, necesitarás:
• Lutron Caseta (funciona sin neutro, pero requiere hub ~$80)
• Instalar nuevo cable (electricista profesional recomendado)
